# 📘 Contexto del Proyecto: INVERSIONES DEIN

## 🧩 Descripción General
El proyecto será un **sistema de préstamos y ahorros** llamado **INVERSIONES DEIN**.

El sistema estará orientado a la gestión financiera operativa de una entidad que maneja préstamos, ahorros, clientes, asesores y caja.

---

## ⚙️ Stack Tecnológico Obligatorio

- **Frontend:** Vue 3 (Composition API)
- **Backend:** Laravel 12

### Librerías y herramientas obligatorias:
- Bootstrap
- Font Awesome
- SweetAlert

### Futuras integraciones:
- Pagos vía móvil en campo

---

## 🎨 Identidad Visual y Diseño

- Nombre: **INVERSIONES DEIN**
- Logo: `public/imágenes/logo.jpg`

### Lineamientos de diseño:
- Estética financiera

## 🔌 Integraciones

Se utilizará API de **DNI / RUC** para obtener:

- Nombres
- Apellidos
- Dirección
- Ubigeo
- Fecha de nacimiento

⚠️ La configuración externa será manejada manualmente, pero el sistema debe estar preparado para consumir esta API.
