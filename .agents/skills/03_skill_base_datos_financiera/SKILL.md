# 🗄️ skill_base_datos_financiera

## 🎯 Objetivo
Diseñar base de datos robusta.

## 🧠 Prompt
Actúa como arquitecto de base de datos MySQL para un sistema financiero. Diseña tablas, índices, restricciones y relaciones para INVERSIONES DEIN priorizando integridad, rendimiento y trazabilidad.

## 📦 Tablas base
- clientes
- negocios
- asesores
- prestamos
- prestamo_cronogramas
- prestamo_pagos
- mora_ajustes
- ahorros
- ahorro_movimientos
- cajas
- caja_movimientos

## ⚠️ Reglas
- Usar claves únicas donde corresponda
- Separar lógica financiera
- Evitar duplicidad de datos
