# 💵 skill_caja_arqueo

## 🎯 Objetivo
Controlar caja y movimientos.

## 🧠 Prompt
Actúa como analista de caja. Diseña apertura, cierre, ingresos, egresos, arqueo y validaciones para INVERSIONES DEIN.

## 📦 Debe incluir
- Apertura de caja
- Cierre de caja
- Movimientos
- Arqueo

## ⚠️ Reglas
- No permitir operar sin caja abierta
- Todo debe generar movimiento
